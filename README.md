# Windows-11-Debloater-Lite
Ez egy Windows 11 Debloater script ami csak a kevésbé fontos elemeket törli le. Ha szeretnél egy olyat ami nagyon kitisztítja a Windows-t, akkor nézd meg a profilomon az *XTREME* verziót. 

Hogyan használd? Letöltöd a kellő .ps1 fájlt, majd nyomsz rá egy jobbklikket. Itt ha Windows 11-en vagy akkor kattints a további lehetőségek megtekintése opcióra. Ha ez megvan, akkor azon belül pedig hogy futtatás Powershell használatával, Windows 10-en pedig egyből lehet a futtatás Powershell-el opcióra kattintani. Amennyiben a forráskódot szeretnéd látni, nyisd meg Powershell ISE-vel, vagy valamilyen szövegszerkesztővel. Fontos még megjegyezni hogy a lent felsorolt programokat le kell venni a tálcáról, illetve meg kell szűnetni a start menüben a kitűzést, és hogy internethez kell legyen csatlakoztatva a géped.
