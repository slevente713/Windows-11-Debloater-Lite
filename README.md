# Windows-11-Debloater-Lite
Ez egy Windows 11 Debloater script ami csak a kevésbé fontos elemeket törli le. Ha szeretnél egy olyat ami nagyon kitisztítja a Windows-t, akkor nézd meg a profilomon az *XTREME* verziót. This is a Windows 11 Debloater script that only removes less important components. If you want one that really cleans up Windows, check out the *XTREME* version on my profile. Dies ist ein Windows 11-Debloater-Skript, das nur die weniger wichtigen Elemente entfernt. Wenn du ein Skript suchst, das Windows gründlich bereinigt, schau dir die *XTREME*-Version in meinem Profil an. これは、重要度の低いコンポーネントのみを削除するWindows 11用デブロータースクリプトです。Windowsを徹底的にクリーンアップしたい場合は、私のプロフィールにある*XTREME*バージョンをご覧ください。Это скрипт для очистки Windows 11, который удаляет только менее важные компоненты. Если вам нужен скрипт, который тщательно очищает Windows, посмотрите версию *XTREME* в моём профиле. 

Hogyan használd? Rámész a *Debloater Lite* fájlra, és kimásolod a kódot. Megnyitod a Powershell-t Admin jogosultsággal, és beilleszted a kódot. Megvárod hogy lefusson, és utánna indísd újra a géped. How do you use it? Click on the *Debloater Lite* file and copy the code. Open PowerShell with administrator privileges and paste the code. Wait for it to run, then restart your computer. Wie wird es verwendet? Klicke auf die Datei *Debloater Lite* und kopiere den Code. Öffne PowerShell mit Administratorrechten und füge den Code ein. Warte, bis der Vorgang abgeschlossen ist, und starte deinen Computer anschließend neu. 使い方は？ *Debloater Lite* ファイルを開き、コードをコピーします。管理者権限でPowerShellを起動し、そのコードを貼り付けます。処理が完了するまで待ち、その後、コンピュータを再起動してください。Как использовать? Нажмите на файл *Debloater Lite* и скопируйте код. Откройте Powershell с правами администратора и вставьте код. Подождите, пока программа завершит работу, а затем перезагрузите компьютер.
